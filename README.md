### 📱 Responsive Breakpoint'ler

* 📱 Mobile: `< 576px`
* 📟 Tablet: `≥ 768px`
* 💻 Desktop: `≥ 992px`
* 🖥️ Large Desktop: `≥ 1200px`
* 🧭 Extra Large: `≥ 1400px`
